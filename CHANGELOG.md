# Change Log

## [1.0.0]

- Refresh dependencies and modernize the TypeScript, ESLint, and test toolchain
- Refactor alignment parsing to use signature help when available with a local fallback parser
- Restore alignment support for plain text and other non-LSP documents
- Add broader automated coverage for JavaScript, TypeScript, TSX, Python, C, C++, and C#

## [0.1.3]

- Add option align left/center/right for decimal/non-decimal
- Add support for python style reference comment line

## [0.1.2]

- Add support for argument type : string, operator, function call

## [0.1.1]

- Add semicolon tolerance

## [0.1.0]

- Initial release
